## ✨ Features

- 🏠 &nbsp;**Home** — Hero section with a warm welcome and quick navigation
- 📅 &nbsp;**Events** — Browse and view upcoming alumni events with countdown timers
- 🤝 &nbsp;**Connections** — Posts and updates to stay connected with your network
- 👥 &nbsp;**Groups** — Explore developer and alumni group listings
- 🖼️ &nbsp;**Gallery** — Cherish memories through a curated photo gallery
- ℹ️ &nbsp;**About** — Meet the team behind the project
- 📬 &nbsp;**Contact** — Reach out easily via the contact page
- 🔐 &nbsp;**Login** — Secure login entry point for alumni members

## 🚀 Getting Started

```bash
# 1. Clone the repository
git clone https://github.com/BhureshParmar1436/Alumni.git

# 2. Navigate into the project
cd Alumni

# 3. Open in browser
open index.html
```

> No build tools or dependencies required — it's pure HTML, CSS & JS.
